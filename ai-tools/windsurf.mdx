---
title: "AI-assisted doc writing in Windsurf with Mintlify skill"
description: "Configure Windsurf's Cascade AI assistant with the Mintlify skill to draft, edit, and maintain documentation with accurate component and style guidance."
---

Windsurf's Cascade AI assistant can help you write and maintain your Mintlify documentation directly inside the editor. By installing the Mintlify skill, you give Cascade access to component references, writing standards, and workflow guidance — so its suggestions stay accurate and consistent with Mintlify conventions.

## Prerequisites

- [Windsurf](https://codeium.com/windsurf) editor installed

## Setup

<Steps>
  <Step title="Open your docs directory in Windsurf">
    Open the root of your documentation repository — the folder that contains `docs.json` — as your Windsurf workspace.
  </Step>

  <Step title="Install the Mintlify skill">
    In the integrated terminal, run:

    ```bash
    npx skills add https://mintlify.com/docs
    ```

    This installs the Mintlify skill, which gives Windsurf's Cascade:

    - Mintlify component reference (`<Steps>`, `<Card>`, `<CodeGroup>`, and more)
    - Writing standards and style guidance
    - Workflow guidance for documentation tasks
  </Step>

  <Step title="Customize project instructions">
    Edit `AGENTS.md` in your project root to add project-specific context:

    - Product terminology and preferred usage (for example, "workspace" not "project")
    - Style preferences specific to your docs
    - Content boundaries — what should and shouldn't be documented

    Cascade reads `AGENTS.md` automatically, so any context you add here shapes every AI suggestion in your project.
  </Step>

  <Step title="Start writing with Cascade">
    Open any MDX file and use Cascade to draft, edit, or refactor documentation. Cascade will apply Mintlify component syntax and your project's style preferences automatically.

    <Tip>
      Ask Cascade to generate a new page, wrap content in a `<Steps>` block, or rewrite a section in active voice — it will follow the Mintlify skill guidance you installed.
    </Tip>
  </Step>
</Steps>
