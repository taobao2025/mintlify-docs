## Get started in minutes

<Steps>
  <Step title="Clone your docs repo">
    During onboarding, Mintlify creates a GitHub repository with your documentation. Clone it locally and open it in your editor.
  </Step>
  <Step title="Install the CLI and preview">
    Install the Mintlify CLI and run `mint dev` to see your docs live at `http://localhost:3000`.
  </Step>
  <Step title="Customize and write">
    Update `docs.json` with your brand colors and navigation. Write pages in MDX using Mintlify's rich component library.
  </Step>
  <Step title="Push to publish">
    Commit and push your changes. The Mintlify GitHub app automatically deploys your docs to production.
  </Step>
</Steps>
